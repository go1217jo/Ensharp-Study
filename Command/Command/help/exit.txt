CMD.EXE 프로그램 (명령 인터프리터)이나 현재 일괄 처리 스크립트를
끝냅니다.

EXIT [/B] [exitCode]

  /B          CMD.EXE 대신 현재 일괄 처리 스크립트를 끝내기 위해
              지정합니다. 외부 일괄 처리 스크립트에서 실행하면
              CMD.EXE를 끝냅니다.

  exitCode    번호를 지정합니다. /B가 지정되면 ERRORLEVEL을
              지정한 번호로 설정합니다. CMD.EXE를 끝내면,
              프로세스 종료 코드를 지정한 번호로 설정합니다.
