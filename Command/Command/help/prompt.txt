cmd.exe 명령 프롬프트를 변경합니다.

PROMPT [text]

  text    새 명령 프롬프트를 지정합니다.

프롬프트는 일반 문자와 다음의 특수 코드로 구성될 수 있습니다.

  $A   &(앰퍼샌드)
  $B   |(파이프)
  $C   ((왼쪽 괄호)
  $D   현재 날짜
  $E   이스케이프 코드(ASCII 코드 27)
  $F   )(오른쪽 괄호)
  $G   >(보다 큼 기호)
  $H   백스페이스(이전 문자 지우기)
  $L   <(보다 작음 기호)
  $N   현재 드라이브
  $P   현재 드라이브 및 경로
  $Q   =(등호 기호)
  $S     (공백)
  $T   현재 시간
  $V   Windows 버전 번호
  $_   캐리지 리턴과 줄 바꿈
  $$   $(달러 기호)

명령 확장을 사용하면 PROMPT 명령은 아래와 같은 추가 포맷의
문자를 지원합니다.

  $+   밀어 넣어지는 각 수준에 한 문자씩, PUSHD 디렉터리 스택 높이에
       따라 0개 이상의 + 기호 문자.

  $M   현재 드라이브 문자에 연결된 원격 이름을 표시하거나
       현재 드라이브가 네트워크 드라이브가 아닐 경우
       빈 문자열을 표시합니다.
