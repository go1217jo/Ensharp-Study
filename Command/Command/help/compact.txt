NTFS 파티션에 있는 파일의 압축을 표시하거나 바꿉니다.

COMPACT [/C | /U] [/S[:디렉터리]] [/A] [/I] [/F] [/Q] [/EXE[:알고리즘]]
        [/CompactOs[:옵션] [/WinDir:디렉터리]] [파일 이름 [...]]

  /C        지정한 파일을 압축합니다. 디렉터리가
            표시되고 /EXE가 지정되지 않은 경우
            나중에 추가된 파일도 압축합니다.
  /U        지정한 파일의 압축을 풉니다. 디렉터리가 표시되고
            나중에 추가된 파일을 압축하지 않습니다.
            /EXE를 지정하면 실행 파일로 압축된 파일만
            압축이 풀립니다. 생략하면 NTFS로 압축된 파일만
            압축이 풀립니다.
  /S        지정된 디렉터리와 모든 하위 디렉터리에서 
            지정한 작업을 실행합니다. 기본 "dir"은
            현재 디렉터리입니다.
  /A        숨김 또는 시스템 특성을 가진 파일을 표시합니다. 
            이 파일은 기본적으로 생략됩니다.
  /I        오류가 발생한 후에도 지정된 작업을 계속 실행합니다.
            오류가 발생할 경우 기본적으로 COMPACT가
            중지됩니다.
  /F        이미 압축된 파일을 포함하여 지정된 모든 파일을 강제로
            압축합니다.  이미 압축된 파일은 기본적으로 
            건너뜁니다.
  /Q        가장 중요한 정보만 보고합니다.
  /EXE      자주 읽지만 수정되지 않는 실행 파일에 최적화된
            압축을 사용합니다. 지원되는 알고리즘은 다음과 같습니다.
            XPRESS4K  (가장 빠름) (기본값)
            XPRESS8K
            XPRESS16K
            LZX       (최대 압축)
  /CompactOs 시스템의 압축 상태를 설정하거나 쿼리합니다.  지원되는 옵션은 다음과 같습니다.
             query  - 시스템의 압축 상태를 쿼리합니다.
             always - 모든 OS 이진 파일을 압축하고 시스템 상태를 압축으로 설정합니다.
                      이 상태는 관리자가 변경할 때까지 유지됩니다.
             never  - 모든 OS 이진 파일 압축을 풀고 시스템 상태를
                      비압축으로 설정합니다. 이 상태는 관리자가 변경할 때까지 유지됩니다.
  /WinDir    오프라인 OS를 쿼리할 때 /CompactOs:query와 함께 사용됩니다. Windows가
             설치된 디렉터리를 지정합니다.
  filename  패턴, 파일 또는 디렉터리를 지정합니다. 

  매개 변수 없이 COMPACT를 사용하면, 현재 디렉터리와 그 안에 있는
  모든 파일의  압축 상태를 표시합니다. 여러 개의 파일 이름과
  와일드카드를 사용할 수 있습니다. 매개 변수 사이에는 공백이
  있어야 합니다.
